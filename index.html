Lugares Seguros

Learning Application Development

Lugares Seguros (Safe Places) was an exciting learning project in which I participated as a Front-end Developer. During this project, I had the opportunity to work on various tasks ranging from design in Figma to implementing functionalities using ReactJS and the Ant Design style library. Additionally, this project marked my first introduction to Git and GitHub, fundamental tools for version control and team collaboration.

My main role in the project involved designing and developing the list of places, the detailed view for each place, and the functionality to delete places from the Front-end area. For design purposes, I utilized Figma, a powerful collaborative design tool, to create intuitive and visually appealing interfaces. Then, by leveraging ReactJS, I was able to translate those designs into functional and reusable components.

Badger

Learning Application Development

In Badger, a learning project I'm currently involved in as a Front-End Developer, I've had the opportunity to apply my skills in ReactJS, Git, JavaScript, HTML, and CSS. My contribution has focused on creating an intuitive form for generating new badges, as well as implementing an interactive preview of each badge.

To challenge my CSS knowledge, I chose not to rely on style libraries and instead worked directly on the design provided by the client. Additionally, we made adjustments to the original design in Figma to enhance the user experience. These achievements have strengthened my proficiency in these technologies, and I'm excited to include this project in my portfolio.
